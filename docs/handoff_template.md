# Handoff Template

Use this at the end of each task so other conversations can continue without guessing.

```text
本次目标：

完成内容：

修改文件：

验证命令：

验证结果：

未解决问题：

影响范围：

风险提醒：

建议下一步交给哪个对话：
```

## Rules

- Be specific about file paths.
- Include the exact validation command when one was run.
- If validation was not run, say why.
- Mention user-confirmation requirements for anything involving push, deploy, secrets, deletion, or production config.
- Do not include secrets, tokens, passwords, or full `.env` content.
